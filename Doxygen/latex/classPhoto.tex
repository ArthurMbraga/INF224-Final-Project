\doxysubsection{Detailed Description}
Represents a photo object derived from the \mbox{\hyperlink{classMultimedia}{Multimedia}} base class. 

The \mbox{\hyperlink{classPhoto}{Photo}} class stores information about a photo, including its name, path, latitude, and longitude. It provides methods to get and set the latitude and longitude, as well as display the photo\textquotesingle{}s information and play the photo using the \char`\"{}imagej\char`\"{} command.
