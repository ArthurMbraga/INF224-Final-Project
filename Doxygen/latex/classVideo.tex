\doxysubsection{Detailed Description}
Represents a video file. 

The \mbox{\hyperlink{classVideo}{Video}} class is a derived class of the \mbox{\hyperlink{classMultimedia}{Multimedia}} class and represents a video file. It provides methods to get and set the duration of the video, display information about the video, and play the video using the mpv player.
