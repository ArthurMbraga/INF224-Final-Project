\doxysubsection{Detailed Description}
Represents a film, which is a type of video with chapters. 

The {\ttfamily \mbox{\hyperlink{classFilm}{Film}}} class inherits from the {\ttfamily \mbox{\hyperlink{classVideo}{Video}}} class and adds the functionality to store and retrieve chapters.
